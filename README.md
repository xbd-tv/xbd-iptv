# xbd-iptv
小板凳 优质 IPTV 节目频道资源

### CND
使用 [jsdelivr gh](https://www.jsdelivr.com/github) CDN 加速

*注意使用以下方式刷新 CDN*
```
curl https://purge.jsdelivr.net/{....地址}  // 刷新CDN
```
